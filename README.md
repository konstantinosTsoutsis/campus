# campus
<h3></h3>
<p>This java application handles CRUD operations for campus managment system while storing data in a MySQL DataBase on localhost.<br>The physical model (DataBase.dll) of the DataBase was build using oracle Data Modeler.<br>Basic operations include : 
<br><ul><li>Insert values</li></ul>
<ul><li>Show the courses</li></ul>
<ul><li>Show the teachers</li></ul>
<ul><li>Show the supervisors</li></ul>
<ul><li>Show if a specific student passed a cousre</li></ul>
<ul><li>Show and delete over duration student</li></ul>

</p>
